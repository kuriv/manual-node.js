# 控制台

执行下面的代码，打印信息。

```javascript
console.log('Hello Node.js!');
console.log('Hello %d %s', 233, 'Node.js!');
console.info('This is an informational message!');
console.warn('This is a warning message!');
console.error('This is an error message!');
```

执行下面的代码，统计代码执行时间。

```javascript
console.time('test');
let arr = [];
for (let i = 0; i < 1000; i++) {
    arr.push(i);
}
console.timeEnd('test');
```

执行下面的代码，分组输出信息。

```javascript
console.group('foo');
console.log('Hello Node.js!');
console.groupEnd();
console.group('bar');
console.log('Hello World!');
console.groupEnd();
```

执行下面的代码，以表格形式输出。

```javascript
let arr = [
    {name: 'Tom', age: 20},
    {name: 'Jerry', age: 18}
];
console.table(arr);
```

执行下面的代码，执行断言测试。

```javascript
let num = 233;
console.assert(num == 233, 'Number should be 233.');
console.assert(num == 666, 'Number should be 233.');
```
