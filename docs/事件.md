# 事件

执行下面的代码，创建并监听指定事件，随后触发事件。

```javascript
const event = require('events');
let emitter = new event();
emitter.on('demo', message => console.log(message));
emitter.emit('demo', 'Hello Node.js!');
emitter.emit('demo', 'Hello World!');
```

执行下面的代码，创建单次监听事件。

```javascript
const event = require('events');
let emitter = new event();
emitter.once('demo', message => console.log(message));
emitter.emit('demo', 'Hello Node.js!');
emitter.emit('demo', 'Hello World!');
```

也可以为指定事件创建多个监听器。

```javascript
const event = require('events');
let emitter = new event();
emitter.on('demo', message => console.log(message.toLowerCase()));
emitter.on('demo', message => console.log(message.toUpperCase()));
emitter.emit('demo', 'Hello Node.js!');
```

执行下面的代码，获取监听器信息。

```javascript
const event = require('events');
let emitter = new event();
emitter.on('demo', message => console.log(message.toLowerCase()));
emitter.on('demo', message => console.log(message.toUpperCase()));
console.log(emitter.listenerCount('demo'));
console.log(emitter.listeners('demo'));
```

执行下面的代码，从指定事件中移除监听器。

```javascript
const event = require('events');
let emitter = new event();
let callback1 = message => console.log(message.toLowerCase());
let callback2 = message => console.log(message.toUpperCase());
emitter.on('demo', callback1);
emitter.on('demo', callback2);
emitter.emit('demo', 'Hello Node.js!');
emitter.removeListener('demo', callback1);
emitter.emit('demo', 'Hello Node.js!');
emitter.removeAllListeners('demo');
emitter.emit('demo', 'Hello Node.js!');
```
