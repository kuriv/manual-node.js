# 缓冲区

执行下面的代码，创建指定字节大小的缓冲区。

```javascript
let buffer = Buffer.alloc(10);
console.log(buffer);
```

执行下面的代码，创建指定字节大小的缓冲区，但缓冲区中可能存在旧的数据。

```javascript
let buffer = Buffer.allocUnsafe(100);
console.log(buffer);
```

执行下面的代码，向缓冲区中写入数据。

```javascript
let buffer = Buffer.alloc(20);
let length = buffer.write('Hello Node.js!');
console.log(buffer);
console.log(length);
```

执行下面的代码，实现缓冲区与字符串的转换。

```javascript
let buffer = Buffer.from('Hello Node.js!');
console.log(buffer);
console.log(buffer.toString());
```

也可以创建一个包含数组的缓冲区。

```javascript
let buffer = Buffer.from([233, 666, 100]);
console.log(buffer);
console.log(...buffer);
```

执行下面的代码，合并缓冲区。

```javascript
let buffer1 = Buffer.from('Hello ');
let buffer2 = Buffer.from('Node.js!');
let concat = Buffer.concat([buffer1, buffer2]);
console.log(concat);
console.log(concat.toString());
```

执行下面的代码，比较两个缓冲区。

```javascript
let buffer1 = Buffer.from('ABC');
let buffer2 = Buffer.from('ABA');
console.log(buffer1.compare(buffer2));
```

执行下面的代码，将一个缓冲区中的内容拷贝到另一个缓冲区。

```javascript
let buffer1 = Buffer.from('Hello ');
let buffer2 = Buffer.from('Node.js!');
buffer2.copy(buffer1);
console.log(buffer1.toString());
console.log(buffer2.toString());
```

执行下面的代码，剪切缓冲区。

```javascript
let buffer = Buffer.from('Hello Node.js!');
console.log(buffer.slice(0, 5).toString());
console.log(buffer.slice(6).toString());
```
