# 定时器

执行下面的代码，设置用于在指定时间后执行一次的回调函数。

```javascript
setTimeout(() => console.log('Hello Node.js!'), 3000);
```

执行下面的代码，取消之前设置的定时器。

```javascript
let timer = setTimeout(() => console.log('Hello Node.js!'), 3000);
clearTimeout(timer);
```

执行下面的代码，设置用于在指定时间后重复执行的回调函数。

```javascript
setInterval(() => console.log('Hello Node.js!'), 3000);
```

执行下面的代码，取消之前设置的重复定时器。

```javascript
let interval = setInterval(() => console.log('Hello Node.js!'), 3000);
clearInterval(interval);
```

执行下面的代码，设置用于在当前事件循环结束时执行的回调函数。

```javascript
setImmediate(() => console.log('Hello Node.js!'));
```

执行下面的代码，取消之前设置的定时器。

```javascript
let immediate = setImmediate(() => console.log('Hello Node.js!'));
clearImmediate(immediate);
```
