# 系统信息

执行下面的代码，获取操作系统信息。

```javascript
const os = require('os');
console.log(os.platform());
console.log(os.type());
console.log(os.arch());
console.log(os.release());
console.log(os.uptime());
```

执行下面的代码，获取内存信息。

```javascript
const os = require('os');
console.log(`${os.totalmem() / 1024 / 1024 / 1024} GB`);
console.log(`${os.freemem() / 1024 / 1024 / 1024} GB`);
```

执行下面的代码，获取 CPU 信息。

```javascript
const os = require('os');
console.log(os.cpus());
console.log(os.loadavg());
```

执行下面的代码，获取网络接口信息。

```javascript
const os = require('os');
console.log(os.networkInterfaces());
```

执行下面的代码，获取用户相关信息。

```javascript
const os = require('os');
console.log(os.homedir());
console.log(os.userInfo());
console.log(os.hostname());
```
