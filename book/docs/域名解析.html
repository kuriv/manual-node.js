域名解析

执行下面的代码，获取当前 DNS 服务器。

const dns = require('dns');
console.log(dns.getServers());

执行下面的代码，解析域名的 IP 地址。

const dns = require('dns');
dns.lookup('github.com', (err, address) => {
    if (err) return console.error(err);
    console.log(address);
});

执行下面的代码，解析不同类型的域名记录。

const dns = require('dns');
dns.resolve('github.com', 'A', (err, addresses) => {
    if (err) return console.error(err);
    console.log(addresses);
});
dns.resolve('github.com', 'MX', (err, addresses) => {
    if (err) return console.error(err);
    console.log(addresses);
});
dns.resolve('github.com', 'TXT', (err, addresses) => {
    if (err) return console.error(err);
    console.log(addresses);
});

执行下面的代码，反向解析域名。

const dns = require('dns');
dns.reverse('8.8.8.8', (err, hostnames) => {
    if (err) return console.error(err);
    console.log(hostnames);
});
