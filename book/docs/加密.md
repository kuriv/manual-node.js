# 加密

执行下面的代码，加密字符串。

```javascript
const crypto = require('crypto');
let hash = crypto.createHash('sha256');
let string = 'Hello Node.js!';
hash.update(string);
console.log(hash.digest('hex'));
```

执行下面的代码，生成密码学安全随机数。

```javascript
const crypto = require('crypto');
console.log(crypto.randomBytes(16).toString('hex'));
```
