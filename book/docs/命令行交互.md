# 命令行交互

执行下面的代码，创建命令行交互式应用程序。

```javascript
const readline = require('readline');
const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
});
rl.question('Hello?\r\n', answer => {
    console.log(answer);
    rl.close();
})
rl.on('close', () => {
    console.log('Bye.');
});
```

执行下面的代码，监听用户输入内容。

```javascript
const readline = require('readline');
const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout
});
rl.on('line', line => {
    if (line.trim() === 'exit') {
        rl.close();
        return;
    }
    console.log(line);
});
rl.on('close', () => {
    console.log('Bye.');
});
```

执行下面的代码，实现命令补全。

```javascript
const readline = require('readline');
const completion = ['exit', 'help', 'status'];
const rl = readline.createInterface({
    input: process.stdin,
    output: process.stdout,
    completer: line => {
        let matches = completion.filter(cmd => cmd.startsWith(line.toLowerCase()));
        return [matches.length ? matches : completion, line];
    }
});
rl.on('line', line => {
    console.log(line);
});
```
